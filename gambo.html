Gambo




Description:


This was a project that I worked on in High School with Omar Rizwan.

We thought it would be fun to build an emulator as means of learning more about computer architecture. We selected the Game Boy because the platform had enough complexity to be interesting while remaining understandable. The Game Boy is based around a modified version of a Zilog Z80, and the system hardware is very well documented.

We wrote the program in Java with the intention of porting it to Android. Much of our code was written with multiple target platforms in mind. We split the codebase into a set of core emulation classes and classes that revolved around platform specific functionality like rendering and input. We later discovered that Java was too slow to run the emulator well on Android. Existing Android Game Boy emulators are mostly written in C using the JNI. However, we were able to use Google Web Toolkit to create a web version (above) with very little additional code due to our modular design.

Our code is available on GitHub.

Skills Used:


Java, Assembly, Software Architecture
