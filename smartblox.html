Smart Blocx



Description:


As part of Penn's xLab, I worked on developing the Smart Blox project, which are building blocks that are coupled with other media in your living room. The system would contain a base board that is in communication with a larger networked system. As blocks are placed on the board, the board maintains a digital representation of the structure in memory and can provide it to whatever device it is connected to, such as a video game.

One goal of this project was to keep the cost of each block as low as possible. Moreover, in order for the device to work well, a solid electrical connection is needed, which means we needed to minimize the number of wires we were running to each block. Our solution was to place a DS2413 in each block. These devices follow the 1-Wire protocol, The base board can keep track of the number of blocks on each node and extrapolate the structure from there. Moreover, the GPIO on the chips allows for switches and LEDs to be embedded in the devices to provide limited interaction with individual blocks.
